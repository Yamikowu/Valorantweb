
**1. 網頁有哪些區塊？** 

我的網頁主要分為幾個核心區塊。最上方是「導覽列區塊 (Navbar)」，可以連結到首頁、牌位 (Rank)、遊戲介紹下拉選單 (包含角色、地圖、槍枝) 以及社群媒體。接著中間有一個「歡迎提示框區塊 (Alert)」，用來簡單介紹網站的主題。最下方則是「圖片輪播區塊 (Carousel)」，展示了三張我覺得很帥的《特戰英豪》遊戲宣傳圖。

**2. 哪些地方用了 `div`？** 

我在網頁的架構中大量使用了 `div` 標籤來包裝與排版不同的區塊。例如：

* 在導覽列裡面，我用 `<div class="container-fluid">` 來整體包裝 Logo 以及導覽列中的連結清單，讓它們能有很好的排版空間。
* 在提示框區域，我用了 `<div class="alert alert-primary">` 將提示文字包裝成一個醒目的長條方塊。
* 在底部的圖片輪播結構裡，每一張參與輪播的獨立圖片，外層也都是使用 `<div class="carousel-item">` 包起來的。

**3. 哪些地方用了 `class` 或 `id`？** 

我在專案裡使用了多種 `class` 和 `id` 來方便我針對特定元素設定 CSS 樣式與排版。

* **Class 範例** ：我使用了 `class="welcome-text"` 來指定提示框內的 `<p>` 標籤段落，把它設定成特戰英豪主題顏色的深色（`#0f1923`）並且加粗字體。其他像是 `class="card-container"` 幫助我把外層容器設定為 Flexbox，讓裡面的角色資料卡片能自動換行排列。
* **ID 範例** ：我幫網頁中間的歡迎提示框設定了一個唯一定位的 `id="welcome-alert"`，這樣我在設定該提示框獨有的紅色外框（`border`）與背景顏色時，就不會意外影響到畫面上其他的 alert 區塊。

**4. 哪一塊有使用 `margin` / `padding` / `border`？** 

在我的 `style.css` 裡面，我有運用 CSS 盒模型 (Box Model) 屬性來調整版面的推擠與外觀視覺：

* **Border (邊框)** ：剛剛提到的提示框（`id="welcome-alert"`），我使用了 `border: 2px solid #ff4655;` 幫它加上了具有特戰英豪標誌性「紅色」的實線外邊框。
* **Padding (內邊距)** ：針對網頁的資料表格（如 `.custom-table th` 等），我使用了 `padding: 10px;`；而提示框也設定了 `padding: 15px;`。這可以讓裡面的文字不要直接貼近內側邊框，留有一點空間增加閱讀的舒適度。
* **Margin (外邊距)** ：我針對整體的 `body` 設定了 `margin: 25px;`，讓網頁整體邊緣與瀏覽器視窗之間都留出一些距離，不會整個排版貼死在瀏覽器邊框上。在卡片元件上（`.card`），我也使用了 `margin-right: 35px;` 讓相鄰的兩張卡片之間保持適當的間距。
